1.2 From Expressions to Statements

1.2.1 Expression

A unit is a unit of code which produces a single value when evaluated

1.2.2 Statement

  • Evaluating a statement can produce a value, and/or has some other effect
  • Every expression is a statement, but not vice versa

1.2.3 Examples of Statements

x = 5
return 10
if x < 5:
    y = 10

Every line is a statement

>>> expr.exaluate({'x': 10})
