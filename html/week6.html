1 Recursion

Recursion is a method tn solve a problem by reducing it to subproblems, then combining the subproblem solutions to solve the original problem

  • subproblem must have the same structure
  • some subproblem can be solved directly

1.1 Base Case

The base case is the simplest case of a problem and solved directly

1.2 Recursive Case

We use recursive when the problem is too hard to solve directly e.g. Calculate the sum of a nested list.

2 Design Recipe for recursive functions

The code template for nested lists is 

def f(obj: Union[int, List]) -> ...:
    if isinstance(obj, int):
	...
    else:
	for sublist in obj:
	    ... f(sublist) ...
