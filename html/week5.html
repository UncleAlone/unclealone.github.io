1 Linked Lists

There are two classes for the implementation of linked lists

  • _Node: allows us to string together elements of a list
  • LinkedList: stores a reference to the first Node in the list

1.1 Nodes

def _Node:
    item: Any
    next: Optional[_Node]

    def __init__(self, item: Any) -> None:
	self.item = item
	self.next = None

1.2 LinkedLists

class LinkedList:
    _first: Optional[_Node]

    def __init__(self) -> None:
	self._first = None

2 Linked-List Insertion

2.1 Prepend

Inserting at the front of a linked list.

  • Change what _first refers to
  • No need to iterate through the linked list
  • Runtime: O(1)

2.2 Append

Inserting at the back of a linked list.

  • Iterate through the linked list to find the last node
  • Runtime: O(n)

2.3 Strategy 1: iterate to the node before the desired position

i = 0
curr = self._first
while not (curr is None or i == index - 1):
    curr = curr.next
    i += 1

2.4 Strategy 2: explicitly track the previous node.

i = 0
prev = None
curr = self._first
while not (curr is None or i == index):
    prev, curr = curr, curr.next
    i += 1

3 Another Linked List Design

Store more info in linked lists:

  • _first: refers to first node
  • _last: refers to the last node
  • _size: refers to the length of the linked list
